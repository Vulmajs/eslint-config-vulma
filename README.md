> Official ESLint rules for VulmaJs

## Usage

```bash
npm i -D eslint-config-vulmajs
# or
yarn add -D eslint-config-vulmajs
```

Add it to your `.eslintrc.js` file:

```json
{
  "extends": [
    "vulmajs"
  ]
}
```
